# Proxy Collector

[🇬🇧 English version](README.md)

Плагин для [exteraGram](https://exteragram.app) и [AyuGram](https://ayugram.one), который собирает прокси из публичных Telegram-каналов, чатов, постов и топиков — и проверяет их доступность через нативный движок Telegram.

---

## Возможности

- 🔍 Сбор прокси из публичных каналов, чатов, конкретных постов и топиков форумов
- ⚡ Проверка прокси тем же нативным методом, который использует сам Telegram
- 🌍 Опциональное определение страны через [ipwho.is](https://ipwho.is)
- 📅 Настраиваемая глубина поиска: от 1 дня до всего времени или вручную
- 📁 Загрузка прокси из файлов на GitHub или локального TXT файла на устройстве
- 🗂 История последних 10 сборов с проверками, сохраняется на диск
- 🌐 Просмотр результатов во встроенном браузере — с фильтрами, экспортом в TG и отправкой в Proxy Vault
- 🛑 Кнопка отмены во время сбора и проверки
- 📌 Быстрый доступ из бокового меню

---

## Два режима

Плагин имеет два независимых режима — со своими источниками, настройками и историей:

| Режим | Описание |
|---|---|
| **Обычные прокси** | Стандартные прокси для повседневного использования |
| **Прокси с обходом белых списков** | Прокси, работающие при активных белых списках |

> ⚠️ Прокси с обходом белых списков необходимо проверять **только при включённых белых списках**. Проверка без активных белых списков даст некорректные результаты — рабочие прокси могут оказаться мёртвыми.

## Поддерживаемые форматы источников

| Формат | Пример |
|---|---|
| Канал | `t.me/proxy_channel` |
| Группа / чат | `t.me/GoodbyeWLChat` |
| Конкретный пост | `t.me/channel/910` |
| Топик форума | `t.me/LowiKForum/10805` |
| Пост в топике | `t.me/LowiKForum/10805/32871` |
| Файл на GitHub | `github.com/user/repo/blob/main/proxies.txt` |
| Локальный TXT файл | `/storage/emulated/0/Download/proxies.txt` |

---

## Поддерживаемые форматы прокси

- `https://t.me/proxy?server=...&port=...&secret=...` (MTProto)
- `tg://proxy?server=...&port=...&secret=...` (MTProto, конвертируется автоматически)
- `https://t.me/socks?server=...&port=...&user=...&pass=...` (SOCKS5)
- `tg://socks?server=...&port=...&user=...&pass=...` (SOCKS5, конвертируется автоматически)
- `Server: ... Port: ... Secret: ...` (текстовый блок)
- `IP:PORT:USER:PASS`
- `USER:PASS@IP:PORT`
- `IP:PORT`

> ⚠️ **Важно:** Для корректной работы с Telegram-источниками необходимо быть **подписанным** на используемые каналы, чаты и топики. Корректная работа без подписки не гарантируется.

VPN-конфиги (`ss://`, `vless://`, `vmess://`, `trojan://` и др.) автоматически отфильтровываются.

Файлы GitHub и локальные TXT поддерживают только ссылки `t.me/proxy?` и `t.me/socks?` (включая `tg://` варианты).

## Страница результатов (браузер)

После сбора результаты открываются во встроенном браузере:

- **Фильтр по пингу** — показать только прокси ниже указанной задержки
- **📲 Добавить в Telegram** — добавить прокси в TG с ограничением по пингу
- **🗄 Отправить в Proxy Vault** — отправить в существующую или новую вкладку [Proxy Vault](https://github.com/zyrus-zi/proxy-vault-plugin)
- **📄 Экспорт в TXT** — отправить TXT файл со всеми прокси в Избранное

## Установка

1. Скачайте последний `.plugin` файл из [Releases](../../releases)
2. Откройте exteraGram или AyuGram
3. Перейдите в **Настройки → Плагины → Установить из файла**
4. Выберите скачанный файл и включите плагин

---

## Требования

- exteraGram `11.12.0`+ или AyuGram (только полная версия — Lite не поддерживает плагины)

## Настройки

У каждого режима свои независимые настройки:

| Настройка | Описание | По умолчанию |
|---|---|---|
| Источники Telegram | Ссылки на каналы, чаты, посты или топики | — |
| Глубина поиска | За какой период искать посты | 7 дней |
| Файлы GitHub | Ссылки на файлы с прокси (каждая с новой строки) | — |
| Локальный TXT файл | Полный путь к TXT файлу на устройстве | — |
| Таймаут проверки | Максимальное время ожидания ответа (сек) | 5 |
| Определять страну | Дополнительный запрос к ipwho.is для живых прокси | ✅ Вкл |
